# Metrox TaxInvoo Flutter MVP

This is a manual Flutter scaffold with MVP screens derived from the wireframes.

Screens included:
- Login (OTP)
- Business setup
- Dashboard
- Invoice flow + GST preview
- Ledger
- Reports
- Expenses

Next steps when Flutter CLI is available:
- Run `flutter create .` inside this folder to generate Android/iOS/web platform files.
- Run `flutter pub get` to fetch dependencies.
